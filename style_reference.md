# Style des images de catégories QQUIZ PRODIGY

## Style visuel identifié :
- **Style** : Cyberpunk / Néon / Futuriste
- **Cadre** : Bordure hexagonale/octogonale avec effet circuit imprimé
- **Couleurs** : Néon violet, cyan, orange, rose sur fond sombre/transparent
- **Icône centrale** : Objet représentatif de la catégorie avec effet glow néon
- **Texte** : Nom de la catégorie en bas, style futuriste/tech
- **Effets** : Lignes de circuit, hexagones décoratifs, lueur néon
- **Format** : PNG avec fond transparent
